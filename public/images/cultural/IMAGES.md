# Cultural History — Image Assets

Drop finished PNG or JPEG files in this folder using the exact filenames below.
The site loads them from `/images/cultural/…` (Vite `public/` directory).

**Current files:** `printing-press.jpg`, `revival-tent.jpg`, `sanitarium.jpg` (~200–380 KB each, 1600×1200 JPEG)

## Recommended export settings

| Setting | Value |
|---------|--------|
| **Dimensions** | **1600 × 1200 px** (4:3) — displays at ~288px height; extra resolution for retina |
| **Alt crop** | 1200 × 900 px minimum if file size is a concern |
| **Format** | JPEG quality 85–90, or WebP |
| **Max file size** | ~400 KB each (target) |
| **Color** | Warm, slightly desaturated; site applies ~10% sepia in CSS |
